Initialization

Button can by initialized by setting one of the following CSS classes:
  • .btn - initializes component and adds just some basic, common button styles
  • .btn--primary - initializes component and adds "primary" styles
  • .btn--secondary - initializes component and adds "primary" styles
When setting on of the .btn--primary or .btn--secondary classes, .btn class will be added implicitly.

Name Description
