Loading distractor & loading spinner

<gh-loading-spinner> and <gh-loading-distractor> are a circular indicators of activity (eg. loading).

Progress spinner

NOTE:
Spinner inside input field will be supported once input fields are in place.

Progress distractor

The <gh-loading-distractor> wraps the spinner and adds a label next to it.

Options & Properties

Name Type Default Description
ng-content string (html content) - Defines the label of the loading distractor

Accessibility

Each <gh-loading-spinner> should be given a meaningful label via aria-label or aria-labelledby. <gh-loading-distractor> will set the aria-labelledby attributes automatically.
