ButtonToggle

This component create a button toggle elements with Dynatrace styling.

Imports

You have to import the DtButtonToggleModule when you want to use the dt-button-toggle:

@NgModule({{"{"}}
  imports: [
    DtButtonToggleModule,
  ],
{{"}"}})
class MyModule {{"{}"}}

Initialization

To apply the dynatrace button toggle, use the <dt-button-toggle> and <dt-button-toggle-item> elements. Example:

Attribute Description
dt-button-toggle Indicate the outer element of the button toggle. The group element can contain multiple <dt-button-toggle-item< elements.
dt-button-toggle-item The element of an individual button.

Options & Properties

Button toggle

Name Type Default Description
[value] T | undefined undefined The current (selected) value
[disabled] boolean | undefined undefined Sets disable state if property is set and the value is truthy or undefined
[tabIndex] number 0 Sets and gets the tabIndex property
(valueChange) event<T> Emits an event when the user selects an other button.
selectValue(T) void Select a specific option (value) programmatically.
clearSelection() void Clears the selected option programmatically.

Button toggle item

Name Type Default Description
<ng-content> The content/text which is displayed inside of the item
[value] T | undefined undefined The associated value of this item
[disabled] boolean | undefined undefined Sets disable state if property is set and the value is truthy or undefined
[tabIndex] number 0 Sets and gets the tabIndex property
[selected] boolean false Sets or gets the selected state of this item
