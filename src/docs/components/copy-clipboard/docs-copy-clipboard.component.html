Copy clipboard

The <dt-copy-clipboard> creates a container including a textarea, input or a label ready to be copied to the clipboard. It is possible to disable the entire copy clipboard container.

Using <dt-copy-clipboard-label> defines the button content (e.g. text like "click here to copy").

Imports

You have to import the DtCopyClipboardModule when you want to use the dt-copy-clipboard:

@NgModule({{"{"}}
  imports: [
  DtCopyClipboardModule,
  ],
{{"}"}})
class MyModule {{"{}"}}

Options

Name Datatype Description
enabled boolean If true (= default) the included button and textarea are disabled, no copy is possible
copied event Callback that is triggered after a successful copy
