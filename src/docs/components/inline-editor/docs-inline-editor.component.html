Basic usage

Editor with two-way data-binding

model: {{ sampleModel1 }} 
<h6 dt-inline-editor

JavaScript API

Using programmatic API to control the component

@ViewChild('sampleEditor2') sampleEditor2: DtInlineEditor;
<h6 #sampleEditor2
  dt-inline-editor
  [(ngModel)]="sampleModel2"></h6>

<button (click)="sampleEditor2.enterEditing()">open editor</button>
<button (click)="sampleEditor2.saveAndQuitEditing()">save changes</button>
<button (click)="sampleEditor2.cancelAndQuitEditing()">cancel changes</button>

Asynchronous tasks

Successful operation

successfulSave () {
  return new Observable<void>((observer) => {
    setTimeout(() => {
      observer.next();
      observer.complete();
    }, 2e3);
  });
}
<h6 dt-inline-editor
  [(ngModel)]="sampleModel3Successful"
  [onSave]="successfulSave"></h6>

Failing operation

failingSave () {
  return new Observable<void>((observer) => {
    setTimeout(() => {
      observer.error();
    }, 2e3);
  });
}
<h6 dt-inline-editor
  [(ngModel)]="sampleModel3Failing"
  [onSave]="failingSave"></h6>
